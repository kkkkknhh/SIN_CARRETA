25   def root(fun, x0, args=(), method='hybr', jac=None, tol=None, callback=None,
26            options=None):
27       r""" 
28       Find a root of a vector function. 
29    
30       Parameters 
31       ---------- 
32       fun : callable 
33           A vector function to find a root of. 
34    
35           Suppose the callable has signature ``f0(x, *my_args, **my_kwargs)``, where 
36           ``my_args`` and ``my_kwargs`` are required positional and keyword arguments. 
37           Rather than passing ``f0`` as the callable, wrap it to accept 
38           only ``x``; e.g., pass ``fun=lambda x: f0(x, *my_args, **my_kwargs)`` as the 
39           callable, where ``my_args`` (tuple) and ``my_kwargs`` (dict) have been 
40           gathered before invoking this function. 
41       x0 : ndarray 
42           Initial guess. 
43       args : tuple, optional 
44           Extra arguments passed to the objective function and its Jacobian. 
45       method : str, optional 
46           Type of solver. Should be one of 
47    
48           - 'hybr'             :ref:`(see here) <optimize.root-hybr>` 
49           - 'lm'               :ref:`(see here) <optimize.root-lm>` 
50           - 'broyden1'         :ref:`(see here) <optimize.root-broyden1>` 
51           - 'broyden2'         :ref:`(see here) <optimize.root-broyden2>` 
52           - 'anderson'         :ref:`(see here) <optimize.root-anderson>` 
53           - 'linearmixing'     :ref:`(see here) <optimize.root-linearmixing>` 
54           - 'diagbroyden'      :ref:`(see here) <optimize.root-diagbroyden>` 
55           - 'excitingmixing'   :ref:`(see here) <optimize.root-excitingmixing>` 
56           - 'krylov'           :ref:`(see here) <optimize.root-krylov>` 
57           - 'df-sane'          :ref:`(see here) <optimize.root-dfsane>` 
58    
59       jac : bool or callable, optional 
60           If `jac` is a Boolean and is True, `fun` is assumed to return the 
61           value of Jacobian along with the objective function. If False, the 
62           Jacobian will be estimated numerically. 
63           `jac` can also be a callable returning the Jacobian of `fun`. In 
64           this case, it must accept the same arguments as `fun`. 
65       tol : float, optional 
66           Tolerance for termination. For detailed control, use solver-specific 
67           options. 
68       callback : function, optional 
69           Optional callback function. It is called on every iteration as 
70           ``callback(x, f)`` where `x` is the current solution and `f` 
71           the corresponding residual. For all methods but 'hybr' and 'lm'. 
72       options : dict, optional 
73           A dictionary of solver options. E.g., `xtol` or `maxiter`, see 
74           :obj:`show_options()` for details. 
75    
76       Returns 
77       ------- 
78       sol : OptimizeResult 
79           The solution represented as a ``OptimizeResult`` object. 
80           Important attributes are: ``x`` the solution array, ``success`` a 
81           Boolean flag indicating if the algorithm exited successfully and 
82           ``message`` which describes the cause of the termination. See 
83           `OptimizeResult` for a description of other attributes. 
84    
85       See also 
86       -------- 
87       show_options : Additional options accepted by the solvers 
88    
89       Notes 
90       ----- 
91       This section describes the available solvers that can be selected by the 
92       'method' parameter. The default method is *hybr*. 
93    
94       Method *hybr* uses a modification of the Powell hybrid method as 
95       implemented in MINPACK [1]_. 
96    
97       Method *lm* solves the system of nonlinear equations in a least squares 
98       sense using a modification of the Levenberg-Marquardt algorithm as 
99       implemented in MINPACK [1]_. 
100   
101      Method *df-sane* is a derivative-free spectral method. [3]_ 
102   
103      Methods *broyden1*, *broyden2*, *anderson*, *linearmixing*, 
104      *diagbroyden*, *excitingmixing*, *krylov* are inexact Newton methods, 
105      with backtracking or full line searches [2]_. Each method corresponds 
106      to a particular Jacobian approximations. 
107   
108      - Method *broyden1* uses Broyden's first Jacobian approximation, it is 
109        known as Broyden's good method. 
110      - Method *broyden2* uses Broyden's second Jacobian approximation, it 
111        is known as Broyden's bad method. 
112      - Method *anderson* uses (extended) Anderson mixing. 
113      - Method *Krylov* uses Krylov approximation for inverse Jacobian. It 
114        is suitable for large-scale problem. 
115      - Method *diagbroyden* uses diagonal Broyden Jacobian approximation. 
116      - Method *linearmixing* uses a scalar Jacobian approximation. 
117      - Method *excitingmixing* uses a tuned diagonal Jacobian 
118        approximation. 
119   
120      .. warning:: 
121   
122          The algorithms implemented for methods *diagbroyden*, 
123          *linearmixing* and *excitingmixing* may be useful for specific 
124          problems, but whether they will work may depend strongly on the 
125          problem. 
126   
127      .. versionadded:: 0.11.0 
128   
129      References 
130      ---------- 
131      .. [1] More, Jorge J., Burton S. Garbow, and Kenneth E. Hillstrom. 
132         1980. User Guide for MINPACK-1. 
133      .. [2] C. T. Kelley. 1995. Iterative Methods for Linear and Nonlinear 
134         Equations. Society for Industrial and Applied Mathematics. 
135         <https://archive.siam.org/books/kelley/fr16/> 
136      .. [3] W. La Cruz, J.M. Martinez, M. Raydan. Math. Comp. 75, 1429 (2006). 
137   
138      Examples 
139      -------- 
140      The following functions define a system of nonlinear equations and its 
141      jacobian. 
142   
143      >>> import numpy as np 
144      >>> def fun(x): 
145      ...     return [x[0]  + 0.5 * (x[0] - x[1])**3 - 1.0, 
146      ...             0.5 * (x[1] - x[0])**3 + x[1]] 
147   
148      >>> def jac(x): 
149      ...     return np.array([[1 + 1.5 * (x[0] - x[1])**2, 
150      ...                       -1.5 * (x[0] - x[1])**2], 
151      ...                      [-1.5 * (x[1] - x[0])**2, 
152      ...                       1 + 1.5 * (x[1] - x[0])**2]]) 
153   
154      A solution can be obtained as follows. 
155   
156      >>> from scipy import optimize 
157      >>> sol = optimize.root(fun, [0, 0], jac=jac, method='hybr') 
158      >>> sol.x 
159      array([ 0.8411639,  0.1588361]) 
160   
161      **Large problem** 
162   
163      Suppose that we needed to solve the following integrodifferential 
164      equation on the square :math:`[0,1]\times[0,1]`: 
165   
166      .. math:: 
167   
168         \nabla^2 P = 10 \left(\int_0^1\int_0^1\cosh(P)\,dx\,dy\right)^2 
169   
170      with :math:`P(x,1) = 1` and :math:`P=0` elsewhere on the boundary of 
171      the square. 
172   
173      The solution can be found using the ``method='krylov'`` solver: 
174   
175      >>> from scipy import optimize 
176      >>> # parameters 
177      >>> nx, ny = 75, 75 
178      >>> hx, hy = 1./(nx-1), 1./(ny-1) 
179   
180      >>> P_left, P_right = 0, 0 
181      >>> P_top, P_bottom = 1, 0 
182   
183      >>> def residual(P): 
184      ...    d2x = np.zeros_like(P) 
185      ...    d2y = np.zeros_like(P) 
186      ... 
187      ...    d2x[1:-1] = (P[2:]   - 2*P[1:-1] + P[:-2]) / hx/hx 
188      ...    d2x[0]    = (P[1]    - 2*P[0]    + P_left)/hx/hx 
189      ...    d2x[-1]   = (P_right - 2*P[-1]   + P[-2])/hx/hx 
190      ... 
191      ...    d2y[:,1:-1] = (P[:,2:] - 2*P[:,1:-1] + P[:,:-2])/hy/hy 
192      ...    d2y[:,0]    = (P[:,1]  - 2*P[:,0]    + P_bottom)/hy/hy 
193      ...    d2y[:,-1]   = (P_top   - 2*P[:,-1]   + P[:,-2])/hy/hy 
194      ... 
195      ...    return d2x + d2y - 10*np.cosh(P).mean()**2 
196   
197      >>> guess = np.zeros((nx, ny), float) 
198      >>> sol = optimize.root(residual, guess, method='krylov') 
199      >>> print('Residual: %g' % abs(residual(sol.x)).max()) 
200      Residual: 5.7972e-06  # may vary 
201   
202      >>> import matplotlib.pyplot as plt 
203      >>> x, y = np.mgrid[0:1:(nx*1j), 0:1:(ny*1j)] 
204      >>> plt.pcolormesh(x, y, sol.x, shading='gouraud') 
205      >>> plt.colorbar() 
206      >>> plt.show() 
207   
208      """
209      def _wrapped_fun(*fargs):
210          """ 
211          Wrapped `func` to track the number of times 
212          the function has been called. 
213          """
214          _wrapped_fun.nfev += 1
215          return fun(*fargs)
216  
217      _wrapped_fun.nfev = 0
218  
219      if not isinstance(args, tuple):
220          args = (args,)
221  
222      meth = method.lower()
223      if options is None:
224          options = {}
225  
226      if callback is not None and meth in ('hybr', 'lm'):
227          warn(f'Method {method} does not accept callback.',
228               RuntimeWarning, stacklevel=2)
229  
230      # fun also returns the Jacobian
231      if not callable(jac) and meth in ('hybr', 'lm'):
232          if bool(jac):
233              fun = MemoizeJac(fun)
234              jac = fun.derivative
235          else:
236              jac = None
237  
238      # set default tolerances
239      if tol is not None:
240          options = dict(options)
241          if meth in ('hybr', 'lm'):
242              options.setdefault('xtol', tol)
243          elif meth in ('df-sane',):
244              options.setdefault('ftol', tol)
245          elif meth in ('broyden1', 'broyden2', 'anderson', 'linearmixing',
246                        'diagbroyden', 'excitingmixing', 'krylov'):
247              options.setdefault('xtol', tol)
248              options.setdefault('xatol', np.inf)
249              options.setdefault('ftol', np.inf)
250              options.setdefault('fatol', np.inf)
251  
252      if meth == 'hybr':
253          sol = _root_hybr(_wrapped_fun, x0, args=args, jac=jac, **options)
254      elif meth == 'lm':
255          sol = _root_leastsq(_wrapped_fun, x0, args=args, jac=jac, **options)
256      elif meth == 'df-sane':
257          _warn_jac_unused(jac, method)
258          sol = _root_df_sane(_wrapped_fun, x0, args=args, callback=callback,
259                              **options)
260      elif meth in ('broyden1', 'broyden2', 'anderson', 'linearmixing',
261                    'diagbroyden', 'excitingmixing', 'krylov'):
262          _warn_jac_unused(jac, method)
263          sol = _root_nonlin_solve(_wrapped_fun, x0, args=args, jac=jac,
264                                   _method=meth, _callback=callback,
265                                   **options)
266      else:
267          raise ValueError(f'Unknown solver {method}')
268  
269      sol.nfev = _wrapped_fun.nfev
270      return sol
271

273  def _warn_jac_unused(jac, method):
274      if jac is not None:
275          warn(f'Method {method} does not use the jacobian (jac).',
276               RuntimeWarning, stacklevel=2)

279  def _root_leastsq(fun, x0, args=(), jac=None,
280                    col_deriv=0, xtol=1.49012e-08, ftol=1.49012e-08,
281                    gtol=0.0, maxiter=0, eps=0.0, factor=100, diag=None,
282                    **unknown_options):
283      """ 
284      Solve for least squares with Levenberg-Marquardt 
285   
286      Options 
287      ------- 
288      col_deriv : bool 
289          non-zero to specify that the Jacobian function computes derivatives 
290          down the columns (faster, because there is no transpose operation). 
291      ftol : float 
292          Relative error desired in the sum of squares. 
293      xtol : float 
294          Relative error desired in the approximate solution. 
295      gtol : float 
296          Orthogonality desired between the function vector and the columns 
297          of the Jacobian. 
298      maxiter : int 
299          The maximum number of calls to the function. If zero, then 
300          100*(N+1) is the maximum where N is the number of elements in x0. 
301      eps : float 
302          A suitable step length for the forward-difference approximation of 
303          the Jacobian (for Dfun=None). If `eps` is less than the machine 
304          precision, it is assumed that the relative errors in the functions 
305          are of the order of the machine precision. 
306      factor : float 
307          A parameter determining the initial step bound 
308          (``factor * || diag * x||``). Should be in interval ``(0.1, 100)``. 
309      diag : sequence 
310          N positive entries that serve as a scale factors for the variables. 
311      """
312      nfev = 0
313      def _wrapped_fun(*fargs):
314          """ 
315          Wrapped `func` to track the number of times 
316          the function has been called. 
317          """
318          nonlocal nfev
319          nfev += 1
320          return fun(*fargs)
321  
322      _check_unknown_options(unknown_options)
323      x, cov_x, info, msg, ier = leastsq(_wrapped_fun, x0, args=args,
324                                         Dfun=jac, full_output=True,
325                                         col_deriv=col_deriv, xtol=xtol,
326                                         ftol=ftol, gtol=gtol,
327                                         maxfev=maxiter, epsfcn=eps,
328                                         factor=factor, diag=diag)
329      sol = OptimizeResult(x=x, message=msg, status=ier,
330                           success=ier in (1, 2, 3, 4), cov_x=cov_x,
331                           fun=info.pop('fvec'), method="lm")
332      sol.update(info)
333      sol.nfev = nfev
334      return sol

337  def _root_nonlin_solve(fun, x0, args=(), jac=None,
338                         _callback=None, _method=None,
339                         nit=None, disp=False, maxiter=None,
340                         ftol=None, fatol=None, xtol=None, xatol=None,
341                         tol_norm=None, line_search='armijo', jac_options=None,
342                         **unknown_options):
343      _check_unknown_options(unknown_options)
344  
345      f_tol = fatol
346      f_rtol = ftol
347      x_tol = xatol
348      x_rtol = xtol
349      verbose = disp
350      if jac_options is None:
351          jac_options = dict()
352  
353      jacobian = {'broyden1': nonlin.BroydenFirst,
354                  'broyden2': nonlin.BroydenSecond,
355                  'anderson': nonlin.Anderson,
356                  'linearmixing': nonlin.LinearMixing,
357                  'diagbroyden': nonlin.DiagBroyden,
358                  'excitingmixing': nonlin.ExcitingMixing,
359                  'krylov': nonlin.KrylovJacobian
360                  }[_method]
361  
362      if args:
363          if jac is True:
364              def f(x):
365                  return fun(x, *args)[0]
366          else:
367              def f(x):
368                  return fun(x, *args)
369      else:
370          f = fun
371  
372      x, info = nonlin.nonlin_solve(f, x0, jacobian=jacobian(**jac_options),
373                                    iter=nit, verbose=verbose,
374                                    maxiter=maxiter, f_tol=f_tol,
375                                    f_rtol=f_rtol, x_tol=x_tol,
376                                    x_rtol=x_rtol, tol_norm=tol_norm,
377                                    line_search=line_search,
378                                    callback=_callback, full_output=True,
379                                    raise_exception=False)
380      sol = OptimizeResult(x=x, method=_method)
381      sol.update(info)
382      return sol

384  def _root_broyden1_doc():
385      """ 
386      Options 
387      ------- 
388      nit : int, optional 
389          Number of iterations to make. If omitted (default), make as many 
390          as required to meet tolerances. 
391      disp : bool, optional 
392          Print status to stdout on every iteration. 
393      maxiter : int, optional 
394          Maximum number of iterations to make. 
395      ftol : float, optional 
396          Relative tolerance for the residual. If omitted, not used. 
397      fatol : float, optional 
398          Absolute tolerance (in max-norm) for the residual. 
399          If omitted, default is 6e-6. 
400      xtol : float, optional 
401          Relative minimum step size. If omitted, not used. 
402      xatol : float, optional 
403          Absolute minimum step size, as determined from the Jacobian 
404          approximation. If the step size is smaller than this, optimization 
405          is terminated as successful. If omitted, not used. 
406      tol_norm : function(vector) -> scalar, optional 
407          Norm to use in convergence check. Default is the maximum norm. 
408      line_search : {None, 'armijo' (default), 'wolfe'}, optional 
409          Which type of a line search to use to determine the step size in 
410          the direction given by the Jacobian approximation. Defaults to 
411          'armijo'. 
412      jac_options : dict, optional 
413          Options for the respective Jacobian approximation. 
414   
415          alpha : float, optional 
416              Initial guess for the Jacobian is (-1/alpha). 
417          reduction_method : str or tuple, optional 
418              Method used in ensuring that the rank of the Broyden 
419              matrix stays low. Can either be a string giving the 
420              name of the method, or a tuple of the form ``(method, 
421              param1, param2, ...)`` that gives the name of the 
422              method and values for additional parameters. 
423   
424              Methods available: 
425   
426              - ``restart``: drop all matrix columns. Has no extra parameters. 
427              - ``simple``: drop oldest matrix column. Has no extra parameters. 
428              - ``svd``: keep only the most significant SVD components. 
429                Takes an extra parameter, ``to_retain``, which determines the 
430                number of SVD components to retain when rank reduction is done. 
431                Default is ``max_rank - 2``. 
432   
433          max_rank : int, optional 
434              Maximum rank for the Broyden matrix. 
435              Default is infinity (i.e., no rank reduction). 
436   
437      Examples 
438      -------- 
439      >>> def func(x): 
440      ...     return np.cos(x) + x[::-1] - [1, 2, 3, 4] 
441      ... 
442      >>> from scipy import optimize 
443      >>> res = optimize.root(func, [1, 1, 1, 1], method='broyden1', tol=1e-14) 
444      >>> x = res.x 
445      >>> x 
446      array([4.04674914, 3.91158389, 2.71791677, 1.61756251]) 
447      >>> np.cos(x) + x[::-1] 
448      array([1., 2., 3., 4.]) 
449   
450      """
451      pass
