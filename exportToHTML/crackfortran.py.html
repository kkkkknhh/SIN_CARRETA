12   Usage of crackfortran: 
13   ====================== 
14   Command line keys: -quiet,-verbose,-fix,-f77,-f90,-show,-h <pyffilename> 
15                      -m <module name for f77 routines>,--ignore-contains 
16   Functions: crackfortran, crack2fortran 
17   The following Fortran statements/constructions are supported 
18   (or will be if needed): 
19      block data,byte,call,character,common,complex,contains,data, 
20      dimension,double complex,double precision,end,external,function, 
21      implicit,integer,intent,interface,intrinsic, 
22      logical,module,optional,parameter,private,public, 
23      program,real,(sequence?),subroutine,type,use,virtual, 
24      include,pythonmodule 
25   Note: 'virtual' is mapped to 'dimension'. 
26   Note: 'implicit integer (z) static (z)' is 'implicit static (z)' (this is minor bug). 
27   Note: code after 'contains' will be ignored until its scope ends. 
28   Note: 'common' statement is extended: dimensions are moved to variable definitions 
29   Note: f2py directive: <commentchar>f2py<line> is read as <line> 
30   Note: pythonmodule is introduced to represent Python module 
31    
32   Usage: 
33     `postlist=crackfortran(files)` 
34     `postlist` contains declaration information read from the list of files `files`. 
35     `crack2fortran(postlist)` returns a fortran code to be saved to pyf-file 
36    
37     `postlist` has the following structure: 
38    *** it is a list of dictionaries containing `blocks': 
39        B = {'block','body','vars','parent_block'[,'name','prefix','args','result', 
40             'implicit','externals','interfaced','common','sortvars', 
41             'commonvars','note']} 
42        B['block'] = 'interface' | 'function' | 'subroutine' | 'module' | 
43                     'program' | 'block data' | 'type' | 'pythonmodule' | 
44                     'abstract interface' 
45        B['body'] --- list containing `subblocks' with the same structure as `blocks' 
46        B['parent_block'] --- dictionary of a parent block: 
47                                C['body'][<index>]['parent_block'] is C 
48        B['vars'] --- dictionary of variable definitions 
49        B['sortvars'] --- dictionary of variable definitions sorted by dependence (independent first) 
50        B['name'] --- name of the block (not if B['block']=='interface') 
51        B['prefix'] --- prefix string (only if B['block']=='function') 
52        B['args'] --- list of argument names if B['block']== 'function' | 'subroutine' 
53        B['result'] --- name of the return value (only if B['block']=='function') 
54        B['implicit'] --- dictionary {'a':<variable definition>,'b':...} | None 
55        B['externals'] --- list of variables being external 
56        B['interfaced'] --- list of variables being external and defined 
57        B['common'] --- dictionary of common blocks (list of objects) 
58        B['commonvars'] --- list of variables used in common blocks (dimensions are moved to variable definitions) 
59        B['from'] --- string showing the 'parents' of the current block 
60        B['use'] --- dictionary of modules used in current block: 
61            {<modulename>:{['only':<0|1>],['map':{<local_name1>:<use_name1>,...}]}} 
62        B['note'] --- list of LaTeX comments on the block 
63        B['f2pyenhancements'] --- optional dictionary 
64             {'threadsafe':'','fortranname':<name>, 
65              'callstatement':<C-expr>|<multi-line block>, 
66              'callprotoargument':<C-expr-list>, 
67              'usercode':<multi-line block>|<list of multi-line blocks>, 
68              'pymethoddef:<multi-line block>' 
69              } 
70        B['entry'] --- dictionary {entryname:argslist,..} 
71        B['varnames'] --- list of variable names given in the order of reading the 
72                          Fortran code, useful for derived types. 
73        B['saved_interface'] --- a string of scanned routine signature, defines explicit interface 
74    *** Variable definition is a dictionary 
75        D = B['vars'][<variable name>] = 
76        {'typespec'[,'attrspec','kindselector','charselector','=','typename']} 
77        D['typespec'] = 'byte' | 'character' | 'complex' | 'double complex' | 
78                        'double precision' | 'integer' | 'logical' | 'real' | 'type' 
79        D['attrspec'] --- list of attributes (e.g. 'dimension(<arrayspec>)', 
80                          'external','intent(in|out|inout|hide|c|callback|cache|aligned4|aligned8|aligned16)', 
81                          'optional','required', etc) 
82        K = D['kindselector'] = {['*','kind']} (only if D['typespec'] = 
83                            'complex' | 'integer' | 'logical' | 'real' ) 
84        C = D['charselector'] = {['*','len','kind','f2py_len']} 
85                                (only if D['typespec']=='character') 
86        D['='] --- initialization expression string 
87        D['typename'] --- name of the type if D['typespec']=='type' 
88        D['dimension'] --- list of dimension bounds 
89        D['intent'] --- list of intent specifications 
90        D['depend'] --- list of variable names on which current variable depends on 
91        D['check'] --- list of C-expressions; if C-expr returns zero, exception is raised 
92        D['note'] --- list of LaTeX comments on the variable 
93    *** Meaning of kind/char selectors (few examples): 
94        D['typespec>']*K['*'] 
95        D['typespec'](kind=K['kind']) 
96        character*C['*'] 
97        character(len=C['len'],kind=C['kind'], f2py_len=C['f2py_len']) 
98        (see also fortran type declaration statement formats below) 
99    
100  Fortran 90 type declaration statement format (F77 is subset of F90) 
101  ==================================================================== 
102  (Main source: IBM XL Fortran 5.1 Language Reference Manual) 
103  type declaration = <typespec> [[<attrspec>]::] <entitydecl> 
104  <typespec> = byte                          | 
105               character[<charselector>]     | 
106               complex[<kindselector>]       | 
107               double complex                | 
108               double precision              | 
109               integer[<kindselector>]       | 
110               logical[<kindselector>]       | 
111               real[<kindselector>]          | 
112               type(<typename>) 
113  <charselector> = * <charlen>               | 
114               ([len=]<len>[,[kind=]<kind>]) | 
115               (kind=<kind>[,len=<len>]) 
116  <kindselector> = * <intlen>                | 
117               ([kind=]<kind>) 
118  <attrspec> = comma separated list of attributes. 
119               Only the following attributes are used in 
120               building up the interface: 
121                  external 
122                  (parameter --- affects '=' key) 
123                  optional 
124                  intent 
125               Other attributes are ignored. 
126  <intentspec> = in | out | inout 
127  <arrayspec> = comma separated list of dimension bounds. 
128  <entitydecl> = <name> [[*<charlen>][(<arrayspec>)] | [(<arrayspec>)]*<charlen>] 
129                        [/<init_expr>/ | =<init_expr>] [,<entitydecl>] 
130   
131  In addition, the following attributes are used: check,depend,note 
132   
133  TODO: 
134      * Apply 'parameter' attribute (e.g. 'integer parameter :: i=2' 'real x(i)' 
135                                     -> 'real x(2)')

300  def openhook(filename, mode):
301      """Ensures that filename is opened with correct encoding parameter. 
302   
303      This function uses charset_normalizer package, when available, for 
304      determining the encoding of the file to be opened. When charset_normalizer 
305      is not available, the function detects only UTF encodings, otherwise, ASCII 
306      encoding is used as fallback. 
307      """
308      # Reads in the entire file. Robust detection of encoding.
309      # Correctly handles comments or late stage unicode characters
310      # gh-22871
311      if charset_normalizer is not None:
312          encoding = charset_normalizer.from_path(filename).best().encoding
313      else:
314          # hint: install charset_normalizer for correct encoding handling
315          # No need to read the whole file for trying with startswith
316          nbytes = min(32, os.path.getsize(filename))
317          with open(filename, 'rb') as fhandle:
318              raw = fhandle.read(nbytes)
319              if raw.startswith(codecs.BOM_UTF8):
320                  encoding = 'UTF-8-SIG'
321              elif raw.startswith((codecs.BOM_UTF32_LE, codecs.BOM_UTF32_BE)):
322                  encoding = 'UTF-32'
323              elif raw.startswith((codecs.BOM_LE, codecs.BOM_BE)):
324                  encoding = 'UTF-16'
325              else:
326                  # Fallback, without charset_normalizer
327                  encoding = 'ascii'
328      return open(filename, mode, encoding=encoding)

331  def is_free_format(fname):
332      """Check if file is in free format Fortran."""
333      # f90 allows both fixed and free format, assuming fixed unless
334      # signs of free format are detected.
335      result = False
336      if Path(fname).suffix.lower() in COMMON_FREE_EXTENSIONS:
337          result = True
338      with openhook(fname, 'r') as fhandle:
339          line = fhandle.readline()
340          n = 15  # the number of non-comment lines to scan for hints
341          if _has_f_header(line):
342              n = 0
343          elif _has_f90_header(line):
344              n = 0
345              result = True
346          while n > 0 and line:
347              if line[0] != '!' and line.strip():
348                  n -= 1
349                  if (line[0] != '\t' and _free_f90_start(line[:5])) or line[-2:-1] == '&':
350                      result = True
351                      break
352              line = fhandle.readline()
353      return result

357  def readfortrancode(ffile, dowithline=show, istop=1):
358      """ 
359      Read fortran codes from files and 
360       1) Get rid of comments, line continuations, and empty lines; lower cases. 
361       2) Call dowithline(line) on every line. 
362       3) Recursively call itself when statement \"include '<filename>'\" is met. 
363      """
364      global gotnextfile, filepositiontext, currentfilename, sourcecodeform, strictf77
365      global beginpattern, quiet, verbose, dolowercase, include_paths
366  
367      if not istop:
368          saveglobals = gotnextfile, filepositiontext, currentfilename, sourcecodeform, strictf77,\
369              beginpattern, quiet, verbose, dolowercase
370      if ffile == []:
371          return
372      localdolowercase = dolowercase
373      # cont: set to True when the content of the last line read
374      # indicates statement continuation
375      cont = False
376      finalline = ''
377      ll = ''
378      includeline = re.compile(
379          r'\s*include\s*(\'|")(?P<name>[^\'"]*)(\'|")', re.I)
380      cont1 = re.compile(r'(?P<line>.*)&\s*\Z')
381      cont2 = re.compile(r'(\s*&|)(?P<line>.*)')
382      mline_mark = re.compile(r".*?'''")
383      if istop:
384          dowithline('', -1)
385      ll, l1 = '', ''
386      spacedigits = [' '] + [str(_m) for _m in range(10)]
387      filepositiontext = ''
388      fin = fileinput.FileInput(ffile, openhook=openhook)
389      while True:
390          try:
391              l = fin.readline()
392          except UnicodeDecodeError as msg:
393              raise Exception(
394                  f'readfortrancode: reading {fin.filename()}#{fin.lineno()}'
395                  f' failed with\n{msg}.\nIt is likely that installing charset_normalizer'
396                  ' package will help f2py determine the input file encoding'
397                  ' correctly.')
398          if not l:
399              break
400          if fin.isfirstline():
401              filepositiontext = ''
402              currentfilename = fin.filename()
403              gotnextfile = 1
404              l1 = l
405              strictf77 = 0
406              sourcecodeform = 'fix'
407              ext = os.path.splitext(currentfilename)[1]
408              if Path(currentfilename).suffix.lower() in COMMON_FIXED_EXTENSIONS and \
409                      not (_has_f90_header(l) or _has_fix_header(l)):
410                  strictf77 = 1
411              elif is_free_format(currentfilename) and not _has_fix_header(l):
412                  sourcecodeform = 'free'
413              if strictf77:
414                  beginpattern = beginpattern77
415              else:
416                  beginpattern = beginpattern90
417              outmess('\tReading file %s (format:%s%s)\n'
418                      % (repr(currentfilename), sourcecodeform,
419                         strictf77 and ',strict' or ''))
420  
421          l = l.expandtabs().replace('\xa0', ' ')
422          # Get rid of newline characters
423          while not l == '':
424              if l[-1] not in "\n\r\f":
425                  break
426              l = l[:-1]
427          # Do not lower for directives, gh-2547, gh-27697, gh-26681
428          is_f2py_directive = False
429          # Unconditionally remove comments
430          (l, rl) = split_by_unquoted(l, '!')
431          l += ' '
432          if rl[:5].lower() == '!f2py':  # f2py directive
433              l, _ = split_by_unquoted(l + 4 * ' ' + rl[5:], '!')
434              is_f2py_directive = True
435          if l.strip() == '':  # Skip empty line
436              if sourcecodeform == 'free':
437                  # In free form, a statement continues in the next line
438                  # that is not a comment line [3.3.2.4^1], lines with
439                  # blanks are comment lines [3.3.2.3^1]. Hence, the
440                  # line continuation flag must retain its state.
441                  pass
442              else:
443                  # In fixed form, statement continuation is determined
444                  # by a non-blank character at the 6-th position. Empty
445                  # line indicates a start of a new statement
446                  # [3.3.3.3^1]. Hence, the line continuation flag must
447                  # be reset.
448                  cont = False
449              continue
450          if sourcecodeform == 'fix':
451              if l[0] in ['*', 'c', '!', 'C', '#']:
452                  if l[1:5].lower() == 'f2py':  # f2py directive
453                      l = '     ' + l[5:]
454                      is_f2py_directive = True
455                  else:  # Skip comment line
456                      cont = False
457                      is_f2py_directive = False
458                      continue
459              elif strictf77:
460                  if len(l) > 72:
461                      l = l[:72]
462              if l[0] not in spacedigits:
463                  raise Exception('readfortrancode: Found non-(space,digit) char '
464                                  'in the first column.\n\tAre you sure that '
465                                  'this code is in fix form?\n\tline=%s' % repr(l))
466  
467              if (not cont or strictf77) and (len(l) > 5 and not l[5] == ' '):
468                  # Continuation of a previous line
469                  ll = ll + l[6:]
470                  finalline = ''
471                  origfinalline = ''
472              else:
473                  r = cont1.match(l)
474                  if r:
475                      l = r.group('line') # Continuation follows ..
476                  if cont:
477                      ll = ll + cont2.match(l).group('line')
478                      finalline = ''
479                      origfinalline = ''
480                  else:
481                      # clean up line beginning from possible digits.
482                      l = '     ' + l[5:]
483                      # f2py directives are already stripped by this point
484                      if localdolowercase:
485                          finalline = ll.lower()
486                      else:
487                          finalline = ll
488                      origfinalline = ll
489                      ll = l
490  
491          elif sourcecodeform == 'free':
492              if not cont and ext == '.pyf' and mline_mark.match(l):
493                  l = l + '\n'
494                  while True:
495                      lc = fin.readline()
496                      if not lc:
497                          errmess(
498                              'Unexpected end of file when reading multiline\n')
499                          break
500                      l = l + lc
501                      if mline_mark.match(lc):
502                          break
503                  l = l.rstrip()
504              r = cont1.match(l)
505              if r:
506                  l = r.group('line')  # Continuation follows ..
507              if cont:
508                  ll = ll + cont2.match(l).group('line')
509                  finalline = ''
510                  origfinalline = ''
511              else:
512                  if localdolowercase:
513                      # only skip lowering for C style constructs
514                      # gh-2547, gh-27697, gh-26681, gh-28014
515                      finalline = ll.lower() if not (is_f2py_directive and iscstyledirective(ll)) else ll
516                  else:
517                      finalline = ll
518                  origfinalline = ll
519                  ll = l
520              cont = (r is not None)
521          else:
522              raise ValueError(
523                  "Flag sourcecodeform must be either 'fix' or 'free': %s" % repr(sourcecodeform))
524          filepositiontext = 'Line #%d in %s:"%s"\n\t' % (
525              fin.filelineno() - 1, currentfilename, l1)
526          m = includeline.match(origfinalline)
527          if m:
528              fn = m.group('name')
529              if os.path.isfile(fn):
530                  readfortrancode(fn, dowithline=dowithline, istop=0)
531              else:
532                  include_dirs = [
533                      os.path.dirname(currentfilename)] + include_paths
534                  foundfile = 0
535                  for inc_dir in include_dirs:
536                      fn1 = os.path.join(inc_dir, fn)
537                      if os.path.isfile(fn1):
538                          foundfile = 1
539                          readfortrancode(fn1, dowithline=dowithline, istop=0)
540                          break
541                  if not foundfile:
542                      outmess('readfortrancode: could not find include file %s in %s. Ignoring.\n' % (
543                          repr(fn), os.pathsep.join(include_dirs)))
544          else:
545              dowithline(finalline)
546          l1 = ll
547      # Last line should never have an f2py directive anyway
548      if localdolowercase:
549          finalline = ll.lower()
550      else:
551          finalline = ll
552      origfinalline = ll
553      filepositiontext = 'Line #%d in %s:"%s"\n\t' % (
554          fin.filelineno() - 1, currentfilename, l1)
555      m = includeline.match(origfinalline)
556      if m:
557          fn = m.group('name')
558          if os.path.isfile(fn):
559              readfortrancode(fn, dowithline=dowithline, istop=0)
560          else:
561              include_dirs = [os.path.dirname(currentfilename)] + include_paths
562              foundfile = 0
563              for inc_dir in include_dirs:
564                  fn1 = os.path.join(inc_dir, fn)
565                  if os.path.isfile(fn1):
566                      foundfile = 1
567                      readfortrancode(fn1, dowithline=dowithline, istop=0)
568                      break
569              if not foundfile:
570                  outmess('readfortrancode: could not find include file %s in %s. Ignoring.\n' % (
571                      repr(fn), os.pathsep.join(include_dirs)))
572      else:
573          dowithline(finalline)
574      filepositiontext = ''
575      fin.close()
576      if istop:
577          dowithline('', 1)
578      else:
579          gotnextfile, filepositiontext, currentfilename, sourcecodeform, strictf77,\
580              beginpattern, quiet, verbose, dolowercase = saveglobals

664  def split_by_unquoted(line, characters):
665      """ 
666      Splits the line into (line[:i], line[i:]), 
667      where i is the index of first occurrence of one of the characters 
668      not within quotes, or len(line) if no such index exists 
669      """
670      assert not (set('"\'') & set(characters)), "cannot split by unquoted quotes"
671      r = re.compile(
672          r"\A(?P<before>({single_quoted}|{double_quoted}|{not_quoted})*)"
673          r"(?P<after>{char}.*)\Z".format(
674              not_quoted="[^\"'{}]".format(re.escape(characters)),
675              char="[{}]".format(re.escape(characters)),
676              single_quoted=r"('([^'\\]|(\\.))*')",
677              double_quoted=r'("([^"\\]|(\\.))*")'))
678      m = r.match(line)
679      if m:
680          d = m.groupdict()
681          return (d["before"], d["after"])
682      return (line, "")

864          elif c == ')':

873  def markoutercomma(line, comma=','):
874      l = ''
875      f = 0
876      before, after = split_by_unquoted(line, comma + '()')
877      l += before
878      while after:
879          if (after[0] == comma) and (f == 0):
880              l += '@' + comma + '@'
881          else:
882              l += after[0]
883              if after[0] == '(':
884                  f += 1
885              elif after[0] == ')':
886                  f -= 1
887          before, after = split_by_unquoted(after[1:], comma + '()')
888          l += before
889      assert not f, repr((f, line, l))
890      return l

897  def appenddecl(decl, decl2, force=1):
898      if not decl:
899          decl = {}
900      if not decl2:
901          return decl
902      if decl is decl2:
903          return decl
904      for k in list(decl2.keys()):
905          if k == 'typespec':
906              if force or k not in decl:
907                  decl[k] = decl2[k]
908          elif k == 'attrspec':
909              for l in decl2[k]:
910                  decl = setattrspec(decl, l, force)
911          elif k == 'kindselector':
912              decl = setkindselector(decl, decl2[k], force)
913          elif k == 'charselector':
914              decl = setcharselector(decl, decl2[k], force)
915          elif k in ['=', 'typename']:
916              if force or k not in decl:
917                  decl[k] = decl2[k]
918          elif k == 'note':
919              pass
920          elif k in ['intent', 'check', 'dimension', 'optional',
921                     'required', 'depend']:
922              errmess('appenddecl: "%s" not implemented.\n' % k)
923          else:
924              raise Exception('appenddecl: Unknown variable definition key: ' +
925                              str(k))
926      return decl

3544 def traverse(obj, visit, parents=[], result=None, *args, **kwargs):
3545     '''Traverse f2py data structure with the following visit function: 
3546  
3547     def visit(item, parents, result, *args, **kwargs): 
3548         """ 
3549  
3550         parents is a list of key-"f2py data structure" pairs from which 
3551         items are taken from. 
3552  
3553         result is a f2py data structure that is filled with the 
3554         return value of the visit function. 
3555  
3556         item is 2-tuple (index, value) if parents[-1][1] is a list 
3557         item is 2-tuple (key, value) if parents[-1][1] is a dict 
3558  
3559         The return value of visit must be None, or of the same kind as 
3560         item, that is, if parents[-1] is a list, the return value must 
3561         be 2-tuple (new_index, new_value), or if parents[-1] is a 
3562         dict, the return value must be 2-tuple (new_key, new_value). 
3563  
3564         If new_index or new_value is None, the return value of visit 
3565         is ignored, that is, it will not be added to the result. 
3566  
3567         If the return value is None, the content of obj will be 
3568         traversed, otherwise not. 
3569         """ 
3570     '''
3571 
3572     if _is_visit_pair(obj):
3573         if obj[0] == 'parent_block':
3574             # avoid infinite recursion
3575             return obj
3576         new_result = visit(obj, parents, result, *args, **kwargs)
3577         if new_result is not None:
3578             assert _is_visit_pair(new_result)
3579             return new_result
3580         parent = obj
3581         result_key, obj = obj
3582     else:
3583         parent = (None, obj)
3584         result_key = None
3585 
3586     if isinstance(obj, list):
3587         new_result = []
3588         for index, value in enumerate(obj):
3589             new_index, new_item = traverse((index, value), visit,
3590                                            parents=parents + [parent],
3591                                            result=result, *args, **kwargs)
3592             if new_index is not None:
3593                 new_result.append(new_item)
3594     elif isinstance(obj, dict):
3595         new_result = dict()
3596         for key, value in obj.items():
3597             new_key, new_value = traverse((key, value), visit,
3598                                           parents=parents + [parent],
3599                                           result=result, *args, **kwargs)
3600             if new_key is not None:
3601                 new_result[new_key] = new_value
3602     else:
3603         new_result = obj
3604 
3605     if result_key is None:
3606         return new_result
3607     return result_key, new_result

3610 def character_backward_compatibility_hook(item, parents, result,
3611                                           *args, **kwargs):
3612     """Previously, Fortran character was incorrectly treated as 
3613     character*1. This hook fixes the usage of the corresponding 
3614     variables in `check`, `dimension`, `=`, and `callstatement` 
3615     expressions. 
3616  
3617     The usage of `char*` in `callprotoargument` expression can be left 
3618     unchanged because C `character` is C typedef of `char`, although, 
3619     new implementations should use `character*` in the corresponding 
3620     expressions. 
3621  
3622     See https://github.com/numpy/numpy/pull/19388 for more information. 
3623  
3624     """
3625     parent_key, parent_value = parents[-1]
3626     key, value = item
3627 
3628     def fix_usage(varname, value):
3629         value = re.sub(r'[*]\s*\b' + varname + r'\b', varname, value)
3630         value = re.sub(r'\b' + varname + r'\b\s*[\[]\s*0\s*[\]]',
3631                        varname, value)
3632         return value
3633 
3634     if parent_key in ['dimension', 'check']:
3635         assert parents[-3][0] == 'vars'
3636         vars_dict = parents[-3][1]
3637     elif key == '=':
3638         assert parents[-2][0] == 'vars'
3639         vars_dict = parents[-2][1]
3640     else:
3641         vars_dict = None
3642 
3643     new_value = None
3644     if vars_dict is not None:
3645         new_value = value
3646         for varname, vd in vars_dict.items():
3647             if ischaracter(vd):
3648                 new_value = fix_usage(varname, new_value)
3649     elif key == 'callstatement':
3650         vars_dict = parents[-2][1]['vars']
3651         new_value = value
3652         for varname, vd in vars_dict.items():
3653             if ischaracter(vd):
3654                 # replace all occurrences of `<varname>` with
3655                 # `&<varname>` in argument passing
3656                 new_value = re.sub(
3657                     r'(?<![&])\b' + varname + r'\b', '&' + varname, new_value)
3658 
3659     if new_value is not None:
3660         if new_value != value:
3661             # We report the replacements here so that downstream
3662             # software could update their source codes
3663             # accordingly. However, such updates are recommended only
3664             # when BC with numpy 1.21 or older is not required.
3665             outmess(f'character_bc_hook[{parent_key}.{key}]:'
3666                     f' replaced `{value}` -> `{new_value}`\n', 1)
3667         return (key, new_value)
